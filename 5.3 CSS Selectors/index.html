CSS Selectors

Applying CSS to Different Parts of HTML

1. The element selector targets elements based on their HTML tag name.

  1. Class selectors target elements based on the value of the class attribute.
  2. ID selectors target elements based on the value of the id attribute.
  3. Attribute selectors target elements based on their attributes and values.
  4. The universal selector targets all elements.

CSS Code

ol {
  margin-left: -40px;
  margin-top: -20px;
  list-style-position: inside;
}

/* Write your CSS below, don't change the rules above. */
p{
  color:red;
}

.note {
  font-size: 20px
}

#id-selector-demo {
  color: green;
}

li[value="4"] {
  color: blue;
}

*{
  text-align: center;
}

CSS Selector Types

Element Selector

Selects an HTML element

Class Selector

Selects an HTML Class

Id Selector

Selects an HTML "Id" The HTML Id CSS Selector shoudl only be used for one element per HTML document. Whereas the Class Selector should be used to select many elements in an HTML Document

Attribute Selector

Selects an element type with a specific attribute, can also be used to select the anti-attribute of an element.

Universal Selector

Selects all elements in an HTML Document
